Developer API Showcase: Movies and News Fetcher

Project 1: Random Movie Fetcher

This project demonstrates how to fetch a random movie using The Movie Database (TMDb) API. By making an API request to the TMDb service, we can retrieve a list of popular movies and display a random selection from that list.

To Access the Movie Application


site-path/movie     ## replace with your actual path. example(http://127.0.0.1:5000/movie)

Project 2: News Fetcher

This project shows how to fetch news articles related to a user-provided query using the NewsAPI. By entering a query and submitting the form, an API request is made to NewsAPI to fetch relevant news articles, which are then displayed on the page.

To Access the NEWS Application


site-path/news    ## replace with your actual path . example(http://127.0.0.1:5000/news)
